a) Uwagi na Temat JS:
-Kiedy powstał --> W 1995 roku.
-Dlaczego jest to język skryptowy --> Ponieważ kod zostaje w formie tekstu.
-Gdzie jest wykonywany JS (klient lub serwer) --> Java Script działa po stronie klienta.
-Dlaczego jest to bezpieczne narzędzie --> Nie może zrobić niczego, co nie jest dozwolone przez interpreter JavaScript przeglądarki lub kompilator JIT.
-Czy jest to język obiektowy , jeśli tak to dlaczego --> Ponieważ nie posiada klas.

b)Przepisz linie kody pod nimi wytłumaczenie:
‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')" ›
……………………………wytłumaczenie……………………………………………………………………………….
type="button" - okresla przycisk.
name="przycisk" - okresla nazwe (id) przycisku.
value="Nowa Strona" - okresla tekst na przycisku.
onclick - po kliknieciu przycisku wykona sie skrypt.


……………………………wytłumaczenie……………………………………………………………………………….

directories=no, --> widocznosc katalogu
menubar=no, --> widocznosc paska menu
height=280, --> wysokosc
width=160, --> szerokosc
top=200, --> odleglosc od gory
left=200 --> odleglosc od lewej
